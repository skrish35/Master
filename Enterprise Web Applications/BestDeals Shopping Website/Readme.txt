For Startup of the Server:

1. Go to the folder where you have installed the apache tomcat server and go to bin folder in command prompt.

2. Then enter startup.bat

For Compilation:

3. Go to the path webapps/csj/web-inf/classes/ in command prompt.

4. Use the command javac -cp (go to lib path)\servlet-api.jar Welcome.java

For Execution:

5. Go to browser and type localhost.

6. Then click on manager app, enter username and pwd and click on csj

7. Then enter /Welcome
