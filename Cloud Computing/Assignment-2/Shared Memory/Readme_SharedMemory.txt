SHARED MEMORY

This folder consists of five files :

1)MyThreadLarge.java
2)onethread.java
2)twothread.java
3)fourthread.java
4)eighthread.java

Steps to Execute the Program :

1) We need to compile the MyThreadLarge.java program first before compiling the others since this class is used in all the other programs.
   javac MyThreadLarge.java

2) After compilation of MyThreadLarge.java, we need to compile and run the other programs.
   
Compilation:
   javac onethread.java
   javac twothread.java
   javac fourthread.java
   javac eighthread.java

Running :
   java onethread
   java twothread
   java fourthread
   java eighthread
